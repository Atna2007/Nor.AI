## ✨ ¿Qué hace Nor.AI?

- 🚀 **Automatización inteligente** — Automatiza procesos como envío de mensajes, registros, cotizaciones y más, sin sistemas complicados.
- 💬 **Mejor atención al cliente** — Responde consultas de forma automática, disponible 24/7 con respuestas personalizadas.
- ⏱️ **Ahorro de tiempo** — Elimina tareas rutinarias para que puedas enfocarte en hacer crecer tu negocio.
- 🔗 **Compatibilidad** — Se conecta con WhatsApp, Google Sheets, CRMs y más sin cambiar tu flujo de trabajo.

## 📦 Uso Local

```bash
# Clona el repositorio
git clone https://github.com/Atna2007/Nor.AI.git

# Abre en tu navegador
# Opción 1: Doble clic en index.html

# Opción 2: Con servidor local
npx serve .
# Luego abre http://localhost:3000
```

---

## 🌐 Deploy en GitHub Pages

1. Ve a **Settings** del repositorio
2. En **Pages** → **Source**: selecciona `main` branch
3. Guarda y espera ~2 minutos
4. Tu sitio estará disponible en: `https://atna2007.github.io/Nor.AI/`
